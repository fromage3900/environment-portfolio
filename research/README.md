# Surreal Architecture — Research Corpus

Human-readable architectural studies that feed the OS layer (`deploy/surreal_os/`).

## How to add a study

1. Copy `_template_study.md` into a style folder (`zen/`, `gothic/`, `surreal/`, …).
2. Fill **Motifs**, **Proportions**, **Rhythms**, **Structural rules**, **Ornament systems**, **Sources**.
3. List **Extracted atoms** — each atom must land in `deploy/surreal_os/schema/architectural_atoms.yaml`.
4. Link the study from `deploy/SURREAL_ARCHITECTURE_RESEARCH.md`.

## Node design cards

Before implementing a new generator kit, fill `_template_node_design.md` and link it from `deploy/SURREAL_ARCH_LOOP_STATE.md`.
